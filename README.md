## TV DATA AGENT
# OVERVIEW
This project shows how to build an agent using databricks, The data includes the viewership and user information to answer business driven questions in plain and simple language. The goal is to help the business to understand the audience behaviours and channel performance. 

## OBJECTIVE
To provide adata driven insights to understand audience demographics 
identify viewing trends 
measure channel performance 
improve viewer engaement

## DATASET OVERVIEW
The User and Viewership tables were joined using the join statement (UserID)
User table contains:
UserID
Name
Surname
Gender
Race
Age
Province
Viewership table contains:
UserID
Channel
RecordDate
Duration

## FEATURES
Questions that the agent can ask
Total viewership?
Province with highest viewership?
Age group with most watch time?
Channels with highest engagement?

## INSIGHT
The average viewing time is at 8.75 minutes and the highest being 15.27 minutes, which tells us that viewers prefer short duration programs.
The 25-34 age group is leading in total watch time, at 42% of the overall viewing time.
SuperSport Live Events channel has the highest viewers at 1.1k followed by the ICC Cricket World cup at 1.0k, suggesting that sport channels are highly favoured by the males at 89% viewership.

## RECOMMENDATIONS
The station should maximise on the high performing channels and spread them across the least engaging days and times.
More focus should be given to the female audience, making visible the channels they prefer.
